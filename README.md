## Matrix Operation
Simple matrix Calculator that can perform the following:
1. Determinant of the matrix.
2. Matrix Transposition.
3. Matrix Multiplication and Addition.
4. Inverse (WIP)
5. Cofactor of each entries (WIP)
6. Adjoint (WIP)
